# Diwali Sales Data Analysis

This project analyzes a Diwali sales dataset using Python to understand customer purchasing behavior.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Female customers contributed the majority of sales.
- The 26–35 age group generated the highest revenue.
- Uttar Pradesh and Maharashtra had the highest sales.
- IT and Healthcare professionals spent the most.

## Project Objective
The objective of this analysis is to identify customer trends and purchasing patterns during the Diwali shopping season.
